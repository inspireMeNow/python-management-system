# 库存管理系统
## 特点
- 界面使用PyQT6实现
- 后端使用mariadb数据库
- 密码使用md5加密
- 使用模块设计
## 已实现的功能
- 使用数据库连接池
- 用户登录注册功能
- 登录注册界面的绘制
- 库存列表界面的绘制
- 库存信息的增删改查功能
- 实现http服务器便于数据访问
- 实现tcp连接的加密保护数据传输
